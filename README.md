# Food-Diary
Personal development project which takes food data off the internet and tracks intake in a user-friendly way.

## Personal Goals
- Get more comfortable with **Typescript**
- Create a **Secure** app
- Create an **Efficient** app
- Learn to work in a **CI/CD** pipeline
- Learn how to create and use **Github actions**
- Pay extra attention to **Code Structure & Readability**

# Backend
- Learn **React**
- Use **Postgresql** with **Raw SQL** to gain a deeper understanding of queries

# Frontend
- Learn **Node.js**
- Learn to use **HTML/CSS** effectively
